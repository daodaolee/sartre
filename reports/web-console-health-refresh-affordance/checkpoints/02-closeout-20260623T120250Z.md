# Closeout: Web Console Health Refresh Affordance

## Summary

Implemented Web Console feedback for Connector-submitted endpoint health:

- Added a manual `Refresh overview` action that reuses the existing overview loader.
- Refresh failures preserve the current overview and show visible failure detail.
- Agent health cards now summarize `health_report` state as passed, warning, blocked, or missing.
- Health report timestamps are shown in the Agent endpoint health panel.
- Connector setup guidance now includes `pnpm --filter @sartre/connector-cli start -- health qa`.

## Boundary Confirmed

- Web Console still consumes Hub overview only.
- No Hub API internals are imported.
- No Connector Core or Connector CLI internals are imported into Web Console.
- No polling, SSE, or realtime stream was added.
